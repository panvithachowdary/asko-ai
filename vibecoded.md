# vibecoded.md

## Vibe Coding Tools Used
- ChatGPT (UI design, debugging, API integration)
- Cursor (code editing and execution)

## Supporting Configurations
- React frontend built using Vite
- Express backend using Node.js
- Groq API (llama-3.1-8b-instant model)
- MongoDB Atlas for storing question and response data
- Environment variables used for secure API key and database connection
